## üÜò Troubleshooting

### App Won't Launch?
```
1. Open Command Prompt
2. Navigate to folder:
   cd c:\Users\brayd\new workspace\windows moniter\ccleaner_clone
3. Run:
   python app.py
4. Look for error messages
```

### Python Not Found?
```
Solution: Install Python from https://python.org
Make sure to check "Add Python to PATH"
```

### Permission Denied Errors?
```
Solution: Run as Administrator
1. Right-click run.bat
2. Select "Run as administrator"
```

### GUI Not Showing?
```
Solution: Check tkinter
Windows: Should be included with Python
Linux: apt-get install python3-tk
Mac: Use Python from python.org (includes tkinter)
```
